# Vehicle Rental Management System (Java)

## 📌 Description
This project is a vehicle rental management system developed in Java.
It allows the management of different types of vehicles and users, as well as the creation and processing of rental operations.

The system was developed as part of an academic project and applies object-oriented programming principles in a structured and modular way.

---

## 🚗 Features
- Registration of different vehicle types (Bicycle, E-Bike, Scooter)
- User hierarchy (Student, Teacher, Non-Teaching Staff, Employee)
- Rental creation and management
- File reading for vehicles and users
- Object serialization for rental data persistence

---

## 🧠 Object-Oriented Concepts Applied
- Inheritance and polymorphism
- Encapsulation
- Class hierarchy design
- Separation of responsibilities
- File handling (File I/O)
- Object serialization

---

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File I/O
- Serialization
